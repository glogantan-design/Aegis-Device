PLTW Senior Engineering Capstone Project

Aegis Ring Portal

A secure remote configuration interface for the Aegis Ring, a discreet personal safety device designed for high-risk situations.

Developed by: Logan Tan & Montgomery Locke
About this Portal:
This interface allows users to update their emergency contact information remotely. Once saved, the Aegis Ring hardware syncs this data via LTE Cat-M1, ensuring that the device remains independent of local smartphone infrastructure during an emergency.
About this Project

ESP32 Core

Dual-core processing with NVRAM persistence.

LTE Cat-M1

Global cellular connectivity via SIM7080G.

GNSS Positioning

High-precision GPS/GLONASS tracking via Adafruit GPS.

Haptic Driver

Precise alert feedback via DRV2605L & LRA.
